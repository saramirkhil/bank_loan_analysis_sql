# Bank Loan Analysis — SQL Project

## Overview
A complete SQL analysis of a real-world bank loan dataset
using MySQL Workbench. Analyzes loan performance, borrower
behavior, and risk classification across two joined tables.

## Business Questions Answered
- KPI 1: Total loan amount issued per year
- KPI 2: Grade & sub-grade wise revolving balance
- KPI 3: Total payment — Verified vs Non-Verified status
- KPI 4: State wise loan status by last credit pull date
- KPI 5: Home ownership vs last payment date & total payment

## SQL Concepts Used
- INNER JOIN (finance_1 & finance_2)
- GROUP BY / ORDER BY
- Aggregate Functions (SUM, ROUND)
- CONCAT & FORMAT for currency formatting
- Multi-table analysis

## Files
- bank_loan_queries_final.sql — Full KPI analysis
- bank_loan_setup.sql — Database and table setup
- Finance_1.csv — Primary loan dataset
- Finance_2.csv — Secondary borrower dataset
